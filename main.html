Question-1: What are the different ways to select an element in the DOM?

the different ways to select an element in the DOM is getElementById(): Select by ID (unique). getElementsByClassName(): Select by class (returns multiple).getElementsByTagName(): Select by tag name (returns multiple).querySelector(): Select the first element matching a CSS selector.querySelectorAll(): Select all elements matching a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The properties innerHTML, innerText, and textContent are all used to access or modify the content of an element in the DOM, but they behave differently in terms of how they handle the content.innerHTML:(includes the HTML tags).innerText: "Hello world!" (only visible text,ignoring HTML tags).textContent: 'Hello world!' (raw text, no HTML tags, includes hidden text if any).

Question-3: What is event delegation in the DOM?

event delegation is about managing events more efficiently by using a parent element to handle events for multiple child elements.

Question-4: What is event bubbling in the DOM?

event bubbling is a built-in feature of the DOM where events propagate up the DOM tree from the target element to the root. This allows you to listen for events at higher levels in the DOM and handle them efficiently.
